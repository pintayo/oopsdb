Set up OopsDB in 3 steps

1

Install OopsDB

If you haven't already, install the CLI globally:

$ npm install -g oopsdb
2

Activate your license

Copy the license key from your email and run:

$ oopsdb activate <your-license-key>

This activates on your current machine. You get 2 activations (e.g. laptop + desktop).

3

Connect your database

Set up your database connection and start backing up:

$ oopsdb init # connect your DB

$ oopsdb watch # auto-backup every 5 min

That's it! Your database is now protected.

Secure plan? Push to cloud

If you're on the Secure plan, you can also push snapshots to immutable cloud storage:

$ oopsdb secure --push
